# WebApp-Actividades
Aplicación web para la gestión de actividades didácticas.
  - Gestión de usuarios
  - Gestión de roles
  - Gestion de actividades
  - Subscripción a actividad
  - Gestión de comentarios
  - Valoración de actividades

Tecnologías de software:
  - PHP Laravel 5.8
  - Bootstrap
  - MySQL
  - HTML
  - CSS
  - Apache

date dev: 2019
